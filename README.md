## ✈️ Flight Price Prediction using Machine Learning

### 📌 Overview

This project focuses on building a machine learning model to predict flight ticket prices based on various factors such as airline, journey details, duration, and number of stops. The goal is to help users and businesses make informed decisions by estimating ticket prices in advance.

---

### 📊 Problem Statement

Flight ticket prices are highly dynamic and influenced by multiple variables. This project aims to:

* Analyze key factors affecting flight prices
* Build a predictive model to estimate ticket costs
* Provide insights into pricing patterns

---

### 🗂️ Dataset Features

The dataset includes:

* Airline
* Date of Journey
* Source & Destination
* Route
* Departure & Arrival Time
* Duration
* Total Stops
* Additional Information
* Price (Target Variable)

---

### 🔍 Approach

#### 1. Data Preprocessing

* Handled missing values in features like route and stops
* Converted date/time into meaningful features
* Encoded categorical variables

#### 2. Exploratory Data Analysis (EDA)

* Visualized relationships between price and key features
* Identified trends based on airlines, stops, and duration
* Used automated profiling for deeper insights

#### 3. Feature Engineering

* Extracted journey day/month
* Derived duration in numerical format
* Created meaningful input features for modeling

#### 4. Model Building

* Applied regression-based machine learning algorithms
* Trained and evaluated models to optimize performance

---

### 📈 Results

* Successfully built a predictive model for flight prices
* Identified major factors influencing ticket costs
* Improved model performance through feature engineering and tuning

---

### 🛠️ Tech Stack

* Python (NumPy, Pandas, Matplotlib, Scikit-learn)
* Data Visualization & EDA
* Machine Learning (Regression Models)
* ydata-profiling

---

### 💡 Key Insights

* Ticket prices vary significantly based on airline and number of stops
* Duration and timing play a major role in pricing
* Feature engineering greatly improves model accuracy

---

### 🎯 Business Impact

* Helps users predict optimal booking prices
* Useful for travel platforms to implement dynamic pricing strategies
* Enables better decision-making using data-driven insights
